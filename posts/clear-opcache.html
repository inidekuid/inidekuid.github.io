Clear OPCACHE?

Pernah tidak kita sudah melakukan update aplikasi tetapi tidak berubah tampilannya? bisa jadi karena

Pendahuluan

Pernah tidak kita sudah melakukan update aplikasi tetapi tidak berubah tampilannya? bisa jadi karena cache php atau kita mengaktifkan opcache dimana tujuannya "meningkatkan performa dan kecepatan eksekusi skrip PHP dengan menyimpan bytecode hasil kompilasi dalam memori bersama (shared memory)" begitu sih yang saya tau. tapi kadang menyebalkan kalau saat demo eh ternyata ndak berubah. maka sepengetahuan saya bisanya reset manual (saat ini artikel ditulis) dari baca artikel php.net kodenya kurang lebih seperti ini.

Contoh Kode

misalkan simpan dengan nama file bersihkan-cacheop.php


<?php
if (function_exists('opcache_reset')) {
    opcache_reset();
    echo "OPcache cleared successfully.";
} else {
    echo "OPcache is not enabled.";
}
?>
